## Задание 1

**Описание:** Напишите три плейбука:
1. Скачать архив, создать папку и распаковать скачанный архив.
2. Установить пакет `tuned`, запустить как демон, добавить в автозагрузку.
3. Изменить приветствие системы (motd) с использованием переменной.

**Этапы решения:**
- Создан плейбук для скачивания архива Kafka с использованием модулей `get_url` и `unarchive`
- Создан плейбук для установки `tuned` через `apt` и управления сервисом через `systemd`
- Создан плейбук для изменения MOTD с использованием переменной `motd_message` и фактов `ansible_hostname`, `ansible_default_ipv4.address`
- Все плейбуки запущены на группе хостов `all_servers` с правами `become: true`

**Результат выполнения:**
- Архив Kafka скачан и распакован в `/opt/kafka`
- Пакет tuned установлен, запущен и добавлен в автозагрузку
- Приветствие MOTD изменено на кастомное с информацией о системе

## Задание 2

## Задание 3

**Описание:** Создайте плейбук, который будет включать в себя одну роль для установки Apache.

**Роль должна:**
1. Установить веб-сервер Apache
2. Сконфигурировать `index.html` с выводом характеристик компьютера (CPU, RAM, размер первого HDD, IP-адрес) через Jinja2-шаблон
3. Реализовать handler для перезапуска Apache
4. Открыть порт 80, запустить сервер и добавить в автозагрузку
5. Проверить доступность веб-сайта (HTTP 200)

**Этапы решения:**
- Создана структура роли `apache` с директориями tasks, handlers, templates, vars
- В tasks реализована установка Apache через `apt`, создание шаблона `index.html.j2`, запуск сервиса через `systemd`
- В handlers реализован перезапуск Apache при изменении конфигурации
- В шаблоне `index.html.j2` использованы факты `ansible_processor_cores`, `ansible_memtotal_mb`, `ansible_devices`, `ansible_default_ipv4.address`
- Добавлена проверка доступности через модуль `uri` с ожидаемым статусом 200
- Порт 80 открыт через `ufw` (при наличии)

**Результат выполнения:**
- Apache установлен на всех веб-серверах
- Веб-страница отображает характеристики каждого сервера
- Handler сработал при изменении шаблона
- Проверка URI подтвердила доступность сайта (HTTP 200)
